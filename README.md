# Monex

Basic collection of monadic types for Elixir. See [docs](https://hexdocs.pm/monex/api-reference.html).

## Installation

The package can be installed as:

  1. Add `monex` to your list of dependencies in `mix.exs`:

    ```elixir
    def deps do
      [{:monex, "~> 0.1.0"}]
    end
    ```
